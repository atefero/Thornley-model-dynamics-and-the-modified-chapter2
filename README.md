# Thornley-model-dynamics-and-the-modified-chapter2
This repository contains the MATLAB code used to generate the results and figures for **Chapter 2** of the PhD thesis:

"Re-engineering enduring mathematical models using new data and new techniques: Thornley's plant biology model."

Chapter 2 focuses on an in-depth study of the **dynamical behaviour of Thornley’s plant carbon–nitrogen allocation model** and a proposed modified version of the model.

## Contents

Main scripts include:
Poster_Graph_dynamics.m
Poster_Minuslitterterms.m
dynamics_Sixbehaviours.m
ZoomInq2.m

MATLAB (tested with recent MATLAB versions: MATLAB R2024b).

## Authors

Ati Rostami, Brodie A. J. Lawson, and Kevin Burrage. “An in-depth study of the dynamics of Thornley's mathematical model in plant biology with a view to an improved model”. In: Journal of Theoretical Biology (2025), p. 112071. doi: https://doi.org/10.1016/j.jtbi.2025.112071
